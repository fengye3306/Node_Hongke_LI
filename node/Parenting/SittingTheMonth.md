# 坐月子  

核心原则

1. 睡觉优先，醒着再吃，别把饭点当硬指标。
2. 产后前几天胃口差，就先走“汤、粥、蛋羹、软饭”路线。
3. 六餐不是吃更多，而是把一天总量拆小，减轻肠胃负担、方便补能。
4. 每天都要有：主食、蛋白、蔬菜、奶/酸奶、少量坚果、足量饮水。
5. 不走极端无油无盐；按家常清淡口味做即可，别太咸，也别清到没法吃

> 月子餐的逻辑  

* **保护牙齿和骨骼**（核心隐藏逻辑）  

宝宝会优先从母体调用钙。**如果一天只吃三顿，血钙波动大，身体就得不断溶解骨骼来补钙**。把**含钙的食物**（如牛奶、豆制品）分散在加餐里，能给骨骼持续供钙，有保护作用。

* **减轻肠胃负担，促进吸收**   

产后胃肠蠕动慢、肌张力弱。**一顿大餐容易引起胀气、不消化。改成六餐，每餐分量小**，身体能更高效地吸收修复身体、制造乳汁所需的微量营养素。
 
* **维持泌乳、血糖的稳定性**   

母乳妈妈在两餐之间最容易饿，**血糖一低**，情绪和奶量都可能受影响。加餐能快速补能，保持泌乳的节奏。   

> 问答，月子餐不能吃油和盐 是吗？为什么？

绝对不是。这是一个流传很广但非常危险的误区。月子里必须吃盐，也必须摄入优质脂肪！  

**为什么“必须吃盐”？**  

1. 产后大量排汗、排尿、排恶露，**身体是正在大量流失钠的状态**。完全不吃盐，会导致水电解质失衡，全身无力，产后本就虚弱，缺钠会让肌肉无力、恶心头晕，甚至引发低钠血症。    
2. **可能造成乳汁里的“隐形缺钠”**，母亲长期极度低钠，理论上可能影响乳汁钠含量。
用量：每天约一小啤酒瓶盖的量（5-6克），和常人差不多即可。  


**为什么“必须有油”？**    

1. 油是身体修复和制造“聪明奶”的核心原料。     
母乳里约一半的热量来自脂肪，其中必需脂肪酸（亚油酸、α-亚麻酸） 是宝宝大脑和视网膜发育的黄金营养，只能从妈妈的饮食中摄取。不吃油，这最关键的部分就断了供。  

2. 容易便秘   
油脂是肠道的天然润滑剂。清淡到无油，加上产后腹部压力改变，会诱发或加重产后便秘。   

3. 维生素吸收不了   
产检时医生让你补的维生素A、D、E、K是脂溶性的，没有油，吃再多也吸收不了，就等于白补了。  

4. 皮肤和伤口修复变慢   
缺乏必需脂肪酸，细胞膜重建会受阻，皮肤干燥，对剖腹产刀口愈合也不利。
  
  
  
正确做法是“**选好油，巧用油**”：  

多吃“好油”：富含不饱和脂肪酸的油，比如亚麻籽油、紫苏油、核桃油、橄榄油、山茶油。麻油（芝麻油） 也是传统月子餐里很推崇的。  

巧搭配：凉拌用亚麻籽油，快炒用橄榄油或山茶油。亚麻籽油怕热，千万别高温爆炒。麻油也别烧到冒烟，香气和营养就都没了。  

自带油脂的食物：坚果、牛油果、深海鱼，这些也都富含好脂肪。    


耐热：山茶油、橄榄油（轻炒）  
不耐高温：亚麻籽油、紫苏油、核桃油（凉拌、温粥淋入）  
麻油温和适合月子，忌高温烧冒烟  
坚果、深海鱼、牛油果 天然优质脂肪来源，搭配合理。  


> 月子餐规范

* **正餐七分饱，加餐垫一下。**   

加餐不是额外多吃，而是把三餐的一部分热量挪过来。比如正餐少喝一碗汤，留到上午10点作为加餐。  


* **干稀搭配**，每顿正餐都有干有稀，保证泌乳所需水分，又不冲淡胃酸。

* **温热易消化**，所有入口食物均需温热，避开生冷硬辣。
