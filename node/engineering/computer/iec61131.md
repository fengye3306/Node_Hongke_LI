


`IEC（国际电工委员会）`的地位类似于IT 世界里的 `ISO` / `IEEE`，通信世界里的 `3GPP`。
`IEC 61131-3`标准是由 IEC制定的“PLC 编程语言国际标准”，是整个工业自动化领域共同遵循的语言规范。  

> IEC 61131-3 在整个体系里的位置  

`IEC 61131`是 Programmable Controllers（可编程控制器），这是一个系列标准，不是单一文档。

* `IEC 61131-1`	通用概念
* `IEC 61131-2`	硬件要求
* `IEC 61131-3`	编程语言 
* `IEC 61131-4`	用户指南
* `IEC 61131-5`	通信

> IEC 61131-3 到底“规定了什么”？  

它只规定五种标准定义「语言」，不规定「系统实现」  

| 语言 | 全称 | 特点 |
|:-----|:-----|:-----|
| LD | Ladder Diagram | 梯形图 |
| FBD | Function Block Diagram | 功能块图 |
| ST | Structured Text | 类 Pascal |
| IL | Instruction List | 类汇编（已废弃） |
| SFC | Sequential Function Chart | 顺序流程图 |
